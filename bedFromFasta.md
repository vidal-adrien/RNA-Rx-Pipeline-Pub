# bedFromFasta.pl

Creates a `.bed` table of the full length of the sequences from a `.fasta` file.

Usage: `bedFromFasta -i <in.fasta>`

Options:
*  `-i FILE`  Path of the input file (expects fasta format).
*  `-o FILE`  Path of the output file.
*  `-h`       Print this help.
